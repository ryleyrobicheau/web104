App overview

This app will allow the user to find the 'best' resort for the day based on filtered requirments like fresh snow, open trails, snow base. Their location will be stored with the option to update as they travel. In addition it will alert the user when heavy snow is forecasted for a particular resort. This app will be built with data from weatherUnlocked ski resort api and the database will be firebase.

10 User stories

1. As the User I would like to be able to look at the snowfall from more an one resort and compare the two

2. As the user I would want the app to be able to let me know ahead of time about a storm

3. As a Traveler, I want to know what resort in a particular area has the best conditions

4. As An event planner, I would like to know where the resorts with the most consistent snowfall are so I can better plan events

5. As a Resort Operator, I would like to view the data for my resort and post it to our website everyday

6. As a Resort Marketing Director, I would like to be able to generate newsletters with the data to send to our subscribers

7. As a Blogger, I would like to be able to compare criteria of different resorts to better rank them and promt them

8. As a User, I want the data to be displayed in a way that is easy to understand

9. As a User, I want the site to be

10. As a User I want to be able to share the data with my friends on social media

Firebase Usage

Firebase will be used to authenciate user login, which will attach their geo location that they entered to the app to find the relevant data.
